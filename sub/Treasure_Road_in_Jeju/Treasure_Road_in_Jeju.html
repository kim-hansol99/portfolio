Treasure Road in Jeju

여행사 홈페이지 제작

첫번째 프로젝트입니다
가상의 제주도 여행사 브랜드를 만들어 제작한 홈페이지 입니다.
브랜드 아이덴티티에 맞게 보물지도의 컨셉으로 디자인하여 사용자들이 여행지를 보물로 느끼게 디자인 하였습니다.

구분 : 개인 프로젝트

제작 기간 : 4주

제작 인원 : 1명

기여 : 100%

사용 : HTML5 / CSS3 / Javascript

COLOR

'보물지도'의 컨셉에 맞춰 클래식한 브라운톤의 #7c432a, #e8c6a2 색상을 포인트 컬러로 사용하였습니다.
기본 컬러는 #fff7eb를 사용 하였고 블루 컬러는 #c8ccd9을 사용 하였습니다.

FONT

CODING

- 미디어쿼리를 사용하여 반응형 웹페이지로 만들어 화면의 크기에 따라 자연스럽게 변형이 될 수 있도록 작업하였습니다.

- 처음 제작한 웹페이지로 시멘틱 태그를 사용하여 HTML구조를 잡았습니다. grid를 이용해 카테고리별로 배치하였습니다.
